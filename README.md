# Netflix Data Visualization Dashboard

## Objective
To create visualizations and tell a story using Netflix dataset.

## Tools Used
- Power BI

## Dataset
- netflix_titles.csv
- cleaned_netflix.csv

## Visualizations
- Movies vs TV Shows
- Top 10 Countries by Content
- Content Growth Over Years
- Top Genres
- Ratings Distribution

## Key Insights
- Movies dominate over TV Shows
- Content increased rapidly after 2015
- USA has highest number of titles
- Drama and Comedy are most common genres
